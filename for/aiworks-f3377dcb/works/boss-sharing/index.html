線 A｜教學作品

企業內訓：我的數位分身，如何當好慣老闆？

對八位跨部門同事的內部分享：把 AI 從「聰明的顧問」教成「直接交辦的員工」。

對象
8 位跨部門同事
PM、工程師、主管、行銷
時長
1 小時簡報
＋約 3 小時實機 demo 與答疑
形式
企業內訓
簡報＋現場實作
主題
本地 AI Agent
與 Context Engineering

為什麼做這堂課

在企業的這段時間，我觀察到一件事：同事們有使用 AI，但是沒發揮出真正的實力。
多數人沒有系統性學過，公司也沒有培訓，就自己打開網頁版摸索，停在一問一答，不太會下 Prompt，把 AI 當成好用一點的 Google 在查。

結果就是，這麼強的工具，大家只用到一點點；問完拿到答案，事情還是自己做，產出零累積。有人甚至覺得「AI 好像沒大家說的那麼神」。
我看了很可惜。我自己用 AI Agent 幾個月，就做出 Sales Copilot 業務自動化工作流、光學知識庫、提案簡報等，很清楚 AI Agent 能帶來多少改變。
落差從來不在工具，在心智模型：大家還停留在詢問「怎麼做」，沒有人在說「我要什麼結果」。

陪伴別人、幫助別人成長，是我一直在做的事。我用得比較多、走得比較前面，就想把我會的、我懂的分享給他們，幫他們把事情用 AI 做得更快更好，把省下的時間放在更有價值的地方。

這堂課具體想做的，就是把大家從「會問 AI 問題」帶到「會用 AI 實際做事」。
核心的設計理念只有一個：撬動大家的心智模型，改變用 AI 的方式。

設計思路

課前先盤點學員樣貌八個人橫跨 PM、工程師、主管、行銷，對 AI 的起點完全不同。
我課前先盤點各職務的日常任務，demo 場景全部取自他們真實的工作：Excel 比對、簡報排版、做網頁看板。
每個人都能在課裡看到自己的案子。

整堂課用對比框架推進，不堆功能顧問 vs 員工、金魚 vs 分身、消費型 vs 飛輪型。
每組對比都在回答同一個問題：你跟 AI 的關係是什麼。
框架記得住，功能清單記不住。

「慣老闆」是刻意挑釁的包裝對 AI 不需要客氣，要學會交辦、驗收、要求重做。
這個詞讓大家笑，也讓大家一秒記住核心姿態：別問怎麼做，說你要什麼結果。

Context 三層金字塔，把抽象變成可操作的分層L1 當下任務、L2 角色背景、L3 累積知識。
網頁版每次都要重貼，本地 Agent 自動載入三層。
這樣講完，「為什麼要用本地 Agent」就不需要再說服了。

一小時講完就進實機簡報只負責換腦袋，剩下約三小時是現場 demo 和答疑，讓每個人帶著自己的任務來試。
開場先放同事的具體成果：我跟他聊了三十分鐘，分享一支影片，週末他就能自己做出東西。
同儕的成功比講師的成功有說服力。

課程架構

  • 開場：不是教你怎麼用 AI，是教你怎麼解放它，附同事的真實成果
  • 我做了什麼：兩個月的三樣產出，光學知識庫、開發看板、Sales Deck
  • 顧問 vs 員工：網頁版 AI 與本地 Agent 的本質差異
  • 金魚 vs 分身：記憶存在對話裡，還是存在你的硬碟裡
  • 消費型 vs 飛輪型：每次問答一次性消耗，還是每次互動都在累積 Context
  • 兩個場景示範：Excel 與簡報，別問怎麼做，說要什麼結果
  • Context Engineering：三層金字塔與 Context 飛輪
  • 工具選擇：Codex、Claude Code、Antigravity，附延伸學習影片清單
  • Live Demo：現場跑一次
教材節錄｜別再問「怎麼做」

不要這樣問：「VLOOKUP 怎麼寫？」

要這樣說：「我有 A 表跟 B 表，幫我比對 B 表多了哪些客戶，標紅色。」AI 幫你寫公式、跑出結果、直接給你最終的 Excel 檔。
以前你會卡住，是因為你問了「怎麼做」；現在你只要說「我要什麼結果」。

教材節錄｜Context 三層金字塔

L1 當下任務：這次要做什麼、輸出格式，讓 AI 知道目標。
L2 角色背景：你是誰、客戶是誰、你的禁忌，讓 AI 用你的口吻。
L3 累積知識：筆記庫、過往專案、行業 know-how，讓 AI 變成你的領域夥伴。
本地 Agent 同時吃三層，而且自動載入。
網頁版每次都要重新介紹自己，本地版是養一個越來越懂你的分身。

AI Agent 時代真正的差距，不是「誰問得好」，是「誰累積的 context 多」。

成果

在職期間我另外輔導行銷同事用 agent 製作行銷素材，原本的作業時間省下五成以上。
內訓的價值不在課堂那四小時，而是課後有人真的用 AI Agent 幫助他提升工作效率。

這堂課教會我的事

教跨部門的混合受眾，框架要統一，案例要分眾。
所有人共用同一套心智模型，但每個人都要在 demo 裡看到自己的工作。
透過切身的案例和工作流，更能感受到AI Agent帶來的成果。
